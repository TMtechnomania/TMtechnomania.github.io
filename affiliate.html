Announcements & Updates (v1.4.0): Automate Your Mobile Searches

Mobile searches on desktop has been blocked by MS, Use this website on your phone to automate your searches: Automate Your Mobile Searches

v1.4.0 has been launched for everyone

This update introduces a brand-new feature allowing you to toggle any tab between mobile and your regular desktop, enabling the use of a simulated mobile device for other websites. This functionality is especially useful for developers, providing a list of over 60+ different devices to choose from. It aids in debugging websites and testing mobile compatibility and responsiveness.

Still confused how to install and use the extension?

Here is a step by step guide for you.

Installation and Setup:

Step 1: Install the Extension
  1. Visit the Chrome Web Store.
  2. Search for "Rewards Search Automator."
  3. Click on "Add to Chrome" and confirm the installation.

Step 2: Accessing the Extension
  1. Locate the extension icon in your browser toolbar.
  2. Click on the icon to open Rewards Search Automator.

Performing Searches (Search Tab)

Step 1: Manual Entry
  1. Navigate to the "Search" tab.
  2. Manually enter the number of desktop and mobile searches.
  3. Specify the minimum and maximum delay (in seconds) between each search.
  4. Click "Search" to perform searches.

Step 2: Predefined Search Buttons
  1. Choose one of the four predefined search buttons to quickly set search parameters.
  2. Specify the minimum and maximum delay (in seconds) between each search.
  3. Click "Search" to perform searches.

Scheduling Searches (Schedule Tab):

Step 1: Accessing the Schedule Tab
  1. Switch to the "Schedule" tab to set up automated search schedules.
  2. Manually enter the number of desktop and mobile searches.
  3. Specify the minimum and maximum delay (in seconds) between each search.
  4. Choose from four predefined schedules: Never, Startup, Every 5 Minutes, and Every 15 Minutes.

    5. NOTE: Understand that "Every 5 Minutes" includes a random delay of 5-7.5 minutes to prevent automation detection. Similarly, "Every 15 Minutes" includes a random delay of 15-20 minutes.

Footer Features:

Important Links:

  1. Access important links in the footer for additional resources or support.

Device Toggle Mode:

  1. Use the toggle switch to enable or disable simulation mode.
  2. Simulation mode helps users test functionality without actual searches.

Mobile Device Name:

  1. View the allotted mobile device name.
  2. Click on it to switch to another mobile device name.

Facing issues in earning points?

Here is a troubleshooting guide for you.

Points Not Collected After Searches

  1. Adjust Delay Settings:

    • Change the minimum delay to 10-12 seconds and the maximum to 15-18 seconds.
    • Perform searches to see if points are collected.

  2. Check Microsoft Points Availability:

    • Verify if Microsoft has slowed down points collection in your region.
    • If yes, wait for 15 minutes before attempting searches again.

  3. Mobile Search Issues:

    • If mobile searches are not yielding points, change the mobile simulation device from the popup.
    • Retry mobile searches.

  4. Mobile Search Not Counting:

    • If mobile searches still don't count, run desktop searches after setting mobile searches to 0.
    • Refer to the "Searching Tab" for instructions on inspecting the tab and simulating iPhone 12.
Popup Not Launching or Opening Install Page

  1. Accept Terms and Conditions:

    • If the popup is not launching or the install page is not opening, ensure you have agreed to the terms and conditions on the install page.

  2. Reinstall Extension:

    • Reinstall the extension if the popup issues persist, as storage might be corrupted.
Points Not Collected on First Search

  1. Explanation:

    • Microsoft requires two additional searches in the beginning to start earning points for both desktop and mobile.
Searches Are Not Performed

  1. Reinstall or Change Number of Searches:

    • Try reinstalling the extension.
    • If searches are still not performed, consider changing the number of searches.
Schedule Not Working

  1. Browser Settings:

    • Check and disable your browser from running in the background.
    • Disable power-saving mode in your browser.

  2. Confirmation for Scheduled Searches:

    • Confirm if you clicked on "Start Now" in the case of every 5 minutes and 15 minutes schedules to initiate searches in the same session.
