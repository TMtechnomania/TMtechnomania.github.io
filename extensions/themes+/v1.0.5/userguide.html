Remove "New Extension" Warning

Chrome might show a warning dialog or ribbon asking if you want to keep the new tab changes.

  • If you see a dialog, click "Keep changes".
  • If you find a ribbon at the bottom, click "Keep".
Hide Default Chrome Footer

To remove the default pencil icon or footer shown by Chrome:

  • Click the standard Customize Chrome button (bottom right).
  • Disable the "Show footer on the newtab page" toggle.

Under the Hood

Themes+ leverages advanced browser technologies to deliver a premium experience without compromising performance:

  • Smart Storage (IndexedDB): We use IndexedDB to locally store your 4K images and large video wallpapers (up to 100MB). This bypasses standard browser limits, ensures zero-latency loading, and guarantees your personal files never leave your device.
  • Manifest Permissions:
    • Storage: Saving preferences and offline assets.
    • System Display/CPU: Real-time system monitoring charts.
    • Search: Integration with active tab media controls.
    • Tabs: Saving and restoring Workspaces.
    • Alarms/Notifications: Reliable background scheduling for timers.

Action Bar

The control center at the bottom of your screen.

Core Controls

  • Tabs Toggle: Opens Tabs Ribbon (Sidebar).
  • Tools Toggle: Opens Tools Dashboard (Overlay).
  • Bookmarks Toggle: Opens Bookmarks Panel (Sidebar).
  • Fullscreen: Toggles immersive mode.

Widgets & Media

  • Weather: Real-time conditions. Impact: Periodic Internet.
  • Media: Controls playback (Spotify, YouTube, SoundCloud, Apple Music, Amazon Music, Twitch, Netflix, Disney+ Hotstar, Vimeo). Perms: 'Tab Capture' req.
  • Clock: Local system time. Settings available.

Shortcuts

Quick access to favorite sites.

  • Add/Edit: Manage in Settings > Action Bar.
  • Impact: Negligible. Icons cached locally.

Smart Actions

  • Upcoming Alarm: Displays next alarm time if scheduled.
  • Timer Status: Shows countdown if timer active.

Zen Mode

Distraction-free focus environment.

  • Block Web: Prevents tab switching.

FamCam

Camera & Flash utilities.

Tools Dashboard

Access a suite of 13 productivity utilities from the Tools menu.

System Stats

Monitor real-time system performance.

  • View: Graphs for CPU, Memory, and Storage.
  • Details: Hover to see specific model info and available capacities.
  • Impact: Low. Queries local browser API every second.

Workspaces

Save and restore session tab groups.

  • Create: Click (+), name your workspace, and select tabs to save.
  • Restore: Click 'Open' to launch all tabs in the group.
  • Impact: Negligible. Stores text data locally.

Calculator

Standard arithmetic usage.

  • Input: Click buttons or type numbers/operators.
  • Shortcuts: Enter to calculate, Esc to clear, Backspace to delete.

Converter

Unit conversion for everyday needs.

  • Step 1: Select type (Length, Weight, Currency).
  • Step 2: Enter value and choose From/To units.
  • Currency: Uses Internet to fetch rates (cached).

QR Generator

Create QR codes instantly.

  • Generate: Type text or URL in the input box.
  • Download: Click 'Download QR' to save the image.
  • Impact: Client-side generation (No internet).

Text Tools

Analyze and manipulate text.

  • Stats: Live count of characters, words, and lines.
  • Actions: Convert Case (Upper/Lower/Title), Trim Whitespace, Copy.

Password Generator

Create strong credentials.

  • Config: Adjust length slider (8-32). Toggle Uppercase, Numbers, Symbols.
  • Impact: Secure (Generated locally, nothing sent).

JSON Formatter

Validate and beautify JSON data.

  • Format: Paste raw JSON, click 'Format' to indent.
  • Minify: Remove whitespace for compacting.

Base64 Helper

Encode and decode data strings.

  • Encode: Type text -> Click Encode -> Get Base64.
  • Decode: Paste Base64 -> Click Decode -> Get text.

Regex Tester

Test regular expressions.

  • Usage: Enter Regex pattern and Test String.
  • Status: Shows 'Match Found' or 'No Match'.

Hash Generator

Generate cryptographic hashes.

  • algo: Select MD5, SHA-1, or SHA-256.
  • Impact: Compute intensive for large texts (Client-side).

Lorem Ipsum

Generate placeholder text.

  • Config: Choose number of paragraphs/sentences.
  • Copy: One-click copy to clipboard.

RSS Reader

Quick feed preview.

  • Load: Enter RSS URL and click Load.
  • Impact: Uses Internet. Parses feed locally.

Feature Modals

Floatable widgets for notes, bookmarks, and time management.

Notes

Capture thoughts and tasks.

  • Create: Click (+). Limit: Max 10 Notes.
  • Pinning: Click the 'Pin' icon to keep note on desktop.
  • Impact: Low. Saved to local storage (max 5MB text).

Bookmarks Panel

Advanced bookmark management.

  • Access: Click the 'Star' toggle in Action Bar.
  • Search: Filter bookmarks, folders, and URLs instantly.
  • Impact: Low. Queries browser index on open.

Tabs Ribbon

Vertical tab management sidebar.

  • Access: Click the 'App' toggle (leftmost) in Action Bar.
  • Merge: Click 'Merge Windows' to consolidate tabs.
  • Impact: Zero background usage. Active only when open.

Timer & Stopwatch

Time management tools.

  • Timer: Select preset (Short/Long Break) or click Edit to type custom time.
  • Audio: Toggle sound alerts on/off.
  • Impact: Minimal. Uses lightweight interval timers.

Alarms

Never miss an event.

  • Add: Single or Recurring (Daily/Weekly). Limit: Max 3 Active Alarms.
  • Perms: Grant "Notifications" for desktop alerts.
  • Impact: Uses Chrome Alarms API (efficient).
