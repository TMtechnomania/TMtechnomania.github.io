Themes+ Essentials

New Tab Extension v1.0.3.
A powerful, customizable, and feature-rich new tab dashboard for optimal productivity.

Core Settings

Theme Personalization

Customize the visual appearance.

  • Theme: Switch between System (default), Light, or Dark mode.
  • UI Scale: Adjust the interface size from 80% (compact) to 120% (large).
  • Font: Choose from curated fonts like Outfit, Inter, Roboto, etc.

Search & Browsing

Search Engine

Quickly switch between multiple providers.

  • Engines: Google, Bing, DuckDuckGo, YouTube, Spotify, Reddit, etc.
  • Engine Chips: Click the search icon to reveal quick-switch chips.
  • Voice Search: Enable microphone integration for voice queries.

Layout Control

Position the search bar where you want it.

  • Vertical: Top, Center, Bottom.
  • Horizontal: Left, Center, Right.
  • Target: Open results in New Tab or Same Tab.

Wallpaper

Background Types

  • Images: High-resolution static backgrounds.
  • Live Video: Loopable video backgrounds.
  • Mono: Minimalist solid color mode.

Collections & Modes

  • Collections: Choose 'Predefined' or 'User Uploaded'.
  • Mode - Random: Surprises you on every load.
  • Mode - Sequence: Cycles through wallpapers in order.
  • Mode - Specific: Lock to a single favorite wallpaper.

Visual Filters

Fine-tune readability and aesthetics.

  • Blur: Soften background focus.
  • Brightness/Contrast: Adjust lighting levels.
  • Grayscale: Black and white mode.

Action Bar

The control center at the bottom of your screen.

Display Settings

  • Auto-Hide: Fade out when not interacting.
  • Style - Centralised: Compact center island.
  • Style - Spread: Split media, tools, and widgets across the bottom.

Widgets

  • Media Player: Controls for Spotify, YouTube (requires permissions).
  • Weather: Real-time conditions (needs API key).
  • Clock/Date: Configurable formats (12h/24h).
  • Shortcuts: Quick links to favorite sites.

Tools Dashboard

System

  • System Stats: CPU, RAM, and Storage usage monitors.
  • Workspaces: Save and restore groups of tabs.

Utilities

  • Calculator: Standard arithmetic.
  • Converter: Unit conversion (Currency, Length, etc.).
  • QR Generator: Create QR codes from text.
  • Password Gen: Create secure, random passwords.

Developer

  • JSON Formatter: Minify/Format JSON.
  • Base64: Encode/Decode strings.
  • Regex Tester: Test regular expressions.
  • Hash Gen: MD5, SHA-256 generators.

Feature Modals

Notes

Rich text editor with checklist support. Notes can be pinned to the desktop.

Bookmarks

Searchable tree view of your browser bookmarks. Right-click to Copy or Share.

Tabs Ribbon

Vertical list of all open tabs/windows. Includes "Merge Windows" to consolidate.

Timer & Stopwatch

Countdown timer with presets (Coffee, Focus) and a lap-capable stopwatch.

Alarms

Schedule one-time or repeating alarms (Daily, Weekdays). Supports desktop notifications.
